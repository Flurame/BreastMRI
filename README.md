# Breast MRI
Это репозиторий моей курсовой по теме Создание модели - ассистента для первичного анализа МРТ груди при диагностике рака молочной железы
========================================================================================================================================
Основные файлы для использования модели:

MRI_main.ipynb - файл для обучения модели, если у вас есть свободные 7 часов можете тоже попробовать обучить модель

MRI_model_link.txt - готовая для использования рабочая модель, датабаза с 1440 мрт груди для обучения проверки модели. Остаётся только загрузить в приложение из следующего файла

MRI_demo_app.ipynb - приложение для анализа фото, можно попробовать случайное из имеющихся 1400, можно загрузить собственное, если модель не прикреплять программа сама напишет небольшую базовую модельку для анализа
